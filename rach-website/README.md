# РАЧ - Российская Ассоциация Челюстно-лицевых хирургов

Современный веб-сайт для Российской Ассоциации Челюстно-лицевых хирургов.

## Технологии
- Laravel 11
- PHP 8.2
- MySQL
- TailwindCSS
- Filament Admin Panel

## Установка
```bash
composer install
npm install
php artisan migrate
php artisan serve --port=12000
```
